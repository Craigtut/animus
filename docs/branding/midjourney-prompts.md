# Animus Symbol — Midjourney Prompts

## Notes
- All prompts use `--ar 1:1` for square app icon format
- Run each prompt on both backgrounds by swapping the color descriptions
- Feel free to remix or combine elements from different results
- Add `--s 250` for more stylized results or `--s 50` for more literal
- Add `--v 6.1` or your preferred version at the end

## Tips for Iteration

**If results are too complex:** Add "extremely minimal, single element, maximum simplicity" to the prompt

**If results are too digital/tech:** Strengthen the --no list and add "hand-crafted quality, analog feeling, natural"

**If results lack warmth:** Add "warm undertones, candlelight quality, human and inviting"

**If results are too literal:** Add "highly abstract, pure form, non-representational"

**If you find a direction you like:** Use `--cref` with the image URL to create variations, or describe what you liked about it in a new prompt

**For testing at icon size:** After finding strong candidates, re-run with: "app icon at small scale, must be recognizable at 32px, bold simple form" to ensure it holds up
